# Sistema de gerenciamento de estoque e vendas   

## Aplicação desenvolvida em Angular + Node para controle de estoque e vendas de uma loja de motos.

Essa parte do projeto de gerenciamento de estoque, se trata da interface, feita em Angular + Node.js, com o objetivo de suprir a demanda de uma aplicação que contivesse a regra de negócio CurvaABC. Para isso foi implementado também, um sistema de vendas completo.
 

*Gerenciamento e autenticação de usuários (com Roles)

*Gerenciamento de itens e categorias

*Gerenciamento de comprador 

*Gerenciamento de vendas e relatórios completos 

Repositório do BackEnd deste projeto: https://github.com/SantosKristhian/PM-02-4-PERIODO

Inventory and Sales Management System
Application developed in Angular + Node.js for managing the inventory and sales of a motorcycle shop.

This part of the inventory management project refers to the frontend, built with Angular and Node.js, designed to meet the need for an application that implements the ABC classification method (Curva ABC). It also includes a complete sales management system.

User management and authentication (with roles)

Item and category management

Buyer management

Sales management and comprehensive reporting
