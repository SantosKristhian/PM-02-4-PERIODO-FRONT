# Sistema de gerenciamento de estoque e vendas   

## Aplicação desenvolvida em Angular + Node para controle de estoque e vendas de uma loja de motos.

\n Essa parte do projeto de gerenciamento de estoque, se trata da interface, feita em Angular + Node.js, com o objetivo de suprir a demanda de uma aplicação que contivesse a regra de negócio CurvaABC. Para isso foi implementado também, um sistema de vendas completo.\n 
\n Repositório do BackEnd deste projeto: \n 
\n https://github.com/SantosKristhian/PM-02-4-PERIODO \n 

*Gerenciamento e autenticação de usuários (com Roles)

*Gerenciamento de itens e categorias

*Gerenciamento de comprador 

*Gerenciamento de vendas e relatórios completos
